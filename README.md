# Totally Not Hacking

A native SwiftUI iPadOS dashboard app that simulates fictional movie-style hacker interfaces. Visual-only — no real hacking, networking, or intrusion.

## Requirements

- Xcode 16+
- iPadOS simulator or device

## Run

Open `Totally Not Hacking.xcodeproj` in Xcode and run the `Totally Not Hacking` scheme.

## How to Use

The dashboard displays a retro-styled terminal interface with three panes:

- **Top pane:** System load averages, breach track logs, and simulated chatter lines
- **Bottom-left pane:** Scrolling pseudo-source code stream
- **Bottom-right pane:** Doom fire thermal visualization

Tap the **theme cycle button** (top-right corner) to switch between 6 visual themes: Classic Green Terminal, Amber CRT, Ice Blue, Red Alert, Phosphor White, and Cyberpunk Neon. The app persists your selected theme between launches.
